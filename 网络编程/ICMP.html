# ICMP 协议

ICMP（Internal Control Message Protocol）互联网控制报文协议，TCP/IP协议簇的一个子协议。对网络连接状态进行判断。

# 发送一个数据包，方便Wireshark抓包
ping -c 1 www.sourcedev.cc

# 总结

  • data 发送什么数据，应答什么数据，0x08 ~ 0x37，数据包固定长度98个字节
  • id 随机，序列号重置，从0开始
